# Security Policy

We currently support the following versions of AutoDocGen with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of AutoDocGen seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. 

Instead, please report it by:
1.  **Private Disclosure**: Email the maintainer at [your-email@example.com] (Optional: add your email here).
2.  **GitHub Private Reporting**: Use the "Report a vulnerability" button in the **Security** tab of the GitHub repository.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please give us reasonable time to investigate and address the issue before making any public disclosure.
